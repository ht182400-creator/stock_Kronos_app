## 4. 关键技术设计

### 4.1 模型加载策略

```python
class KronosService:
    def __init__(self):
        self._model = None
        self._tokenizer = None
        self._predictor = None
        self._lock = asyncio.Lock()
    
    async def load_model(self, model_key: str, device: str):
        """异步加载模型，防止并发加载"""
        async with self._lock:
            if self._predictor is not None:
                return  # 已加载
            
            tokenizer = KronosTokenizer.from_pretrained(...)
            model = Kronos.from_pretrained(...)
            self._predictor = KronosPredictor(model, tokenizer, device)
```

### 4.2 异步任务队列

```python
class PredictionTask:
    def __init__(self, task_id: str, params: PredictionParams):
        self.task_id = task_id
        self.params = params
        self.status = "pending"
        self.progress = 0
    
    async def execute(self):
        """在后台线程执行（避免阻塞事件循环）"""
        loop = asyncio.get_event_loop()
        result = await loop.run_in_executor(
            None,  # ThreadPoolExecutor
            self._predict_sync
        )
        return result
```

## 5. 错误处理设计

### 5.1 错误码体系

| 错误码 | 说明 | HTTP 状态 |
|--------|------|-----------|
| ERR_MODEL_NOT_LOADED | 模型未加载 | 400 |
| ERR_MODEL_LOAD_FAILED | 模型加载失败 | 500 |
| ERR_FILE_NOT_FOUND | 文件不存在 | 404 |
| ERR_FILE_FORMAT_INVALID | 文件格式错误 | 400 |
| ERR_DATA_INSUFFICIENT | 数据不足 | 400 |
| ERR_PREDICTION_FAILED | 预测失败 | 500 |
| ERR_TASK_NOT_FOUND | 任务不存在 | 404 |
| ERR_QUEUE_FULL | 队列已满 | 503 |
