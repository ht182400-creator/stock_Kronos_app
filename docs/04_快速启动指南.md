## 快速启动

### 第一步：安装后端依赖

```powershell
cd d:/Work_Area/AI/stock_Kronos_app/backend
pip install -r requirements.txt
```

### 第二步：安装前端依赖

```powershell
cd d:/Work_Area/AI/stock_Kronos_app/frontend
npm install
```

### 第三步：启动后端服务

```powershell
cd d:/Work_Area/AI/stock_Kronos_app/backend
python -m uvicorn main:app --reload --port 8000
```

### 第四步：启动前端服务（新终端）

```powershell
cd d:/Work_Area/AI/stock_Kronos_app/frontend
npm run dev
```

### 第五步：访问应用

打开浏览器访问 **http://localhost:5173**

## 想法评估总结

| 你的想法 | 评估 | 说明 |
|---------|------|------|
| Kronos-mini + CPU | ✅ 最佳选择 | 4.1M 参数，CPU 推理约 5-15 秒，更快更轻量 |
| FastAPI | ✅ 正确 | 比 Flask 更好：自动文档、类型安全、异步 |
| React + ECharts | ✅ 正确 | ECharts K线图性能优秀 |
| Dashboard 布局 | ✅ 合理 | 符合金融应用习惯 |
| CSV 导出 | ✅ 必要 | 分析师必备功能 |

## 额外优化建议

- 任务队列 - 防止并发压垮 CPU（已实现）
- 异步预测 - 避免 HTTP 超时（已实现）
- 进度推送 - 实时显示推理进度（已实现）
- 结果缓存 - 避免重复计算（待实现）

## 核心 API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/models/load | 加载模型 |
| POST | /api/data/upload | 上传 CSV |
| POST | /api/predict/async | 异步预测 |
| GET | /api/predict/{id} | 查询结果 |

API 文档：http://localhost:8000/docs
